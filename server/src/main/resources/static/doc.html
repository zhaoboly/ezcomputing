Introduction

EZComputing.org provides a platform to write Javascript codes which can be distributed to a connected worker computer and finish the processing there.

Core Concept

Run Javascript everywhere

The worker software was written using Java and Graalvm. It can received the task from the server and execute the Javascript codes in the task. We have successfully installed the java-based worker system on various systems, including Windows, Mac, Linux. even on Raspberry Pi.
